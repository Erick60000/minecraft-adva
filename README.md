### ✨ Características Principales
* **Optimización de FPS:** Versiones de Minecraft modificadas para máximo rendimiento.
* **Soporte 4D:** Compatibilidad total con geometrías avanzadas y scripts.
* **Interfaz Moderna:** Panel de control rápido y estético.
* **Google Auth:** Sistema de cuentas integrado para usuarios VIP.

---

### 🛠️ Estructura del Proyecto
- `index.html`: Estructura principal y secciones de descarga.
- `style.css`: Diseño avanzado con efectos de cristal y neón.
- `script.js`: El "cerebro" (Modales, Animaciones y Google Login).
- `assets/`: Galería de imágenes y recursos visuales.

### 📥 Instalación para Desarrolladores
Si quieres probarlo localmente usando **Termux**:
1. Clona el repo: `git clone https://github.com/Erick60000/minecraft-adva.git`
2. Entra a la carpeta: `cd minecraft-adva`
3. Abre el index en tu navegador.
